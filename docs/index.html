One-tap permissions for Claude Code

A minimal, keyboard-friendly popup that replaces terminal permission prompts. Navigate with arrow keys, confirm with Enter. That's it.

Built for speed

🎹

Keyboard-First

Arrow keys to navigate, Enter to confirm, Esc to reject. No mouse needed.

⚡

Instant Response

Transparent, always-on-top window appears instantly when Claude Code needs permission.

🔒

Smart Whitelisting

Allow Always adds tools to a session whitelist. No repeated prompts for trusted tools.

🎵

Satisfying Feedback

Mario-style sound effects for every action. Allow, reject — each has its own tune.

🖥️

Cross-Platform

Native support for macOS and Windows. Unix sockets and Named Pipes under the hood.

🪟

Dynamic Island UI

Minimal pill shape when idle, smooth expansion when active. Inspired by iPhone's Dynamic Island.

How it works

⚡ Claude Code triggers a tool
🔌 Hook sends event via IPC
⌨️ Keyboard pops up
✅ Decision sent back
